# CompanyEmployees

API built in .NET 7 and demonstrating the following:

* Configuration
* Logging
* Repository Pattern
* Service Layer
* CRUD Operations
* Parameter Validation
* Asynchronous Code
* Action Filters
* Global Error Handling
* Paging
* Filtering
* Searching
* Sorting
* Data Shaping
* Caching
* JWT and Identity
* Documenting With Swagger
